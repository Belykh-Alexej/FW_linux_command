Основное, системное.
sudo
sudo Sudo (substitute user and do, подменить пользователя и выполнить ) — выполнение команд от имени root или другого пользователя.
apt
apt Менеджер пакетов с интерфейсом командной строки. Он предоставляет команды для поиска и управления, а также запросов информации о пакетах.
sudo !! Команда дублирует пред. идущую команду, в которой забыли прописать sudo, пример : sudo apt-get install alsa , команда выдаст ошибку прав запуска/досупа. Чтоб не повторять всю команду (sudo apt-get install alsa) второй раз, достаточно написать "sudo !!"

Создание команд
alias info=ls Создание команды на 1 сеанс до перезагрузки (команда ls будет выполняться при введении info в консоль).
alias viber='/opt/viber/Viber %u' Создание своей команды для запуска вайбер, будет работать только до перезагрузки линукс.
gedit ~/.bash_aliases !!!Создание постоянной команды!!!

Удаление/чистка
sudo apt-get remove alsa Удаление установленной программы на примере alsa
sudo apt autoclean Чистка системы. Очищает локальный репозиторий от извлеченных файлов пакетов, но удаляет только те файлы, которые больше не могут быть загружены и практически бесполезны. Это помогает сохранить ваш кэш от переполнения.
sudo apt autoremove Проверяет системы и чистка от устаревших ядер. Удаляет пакеты, которые были установлены автоматически, поскольку они требовались для некоторых других пакетов, но после удаления этих пакетов они больше не нужны. Иногда при обновлении рекомендуется выполнить эту команду.

    ls
    ls ls Команда показывает каталоги (папки в виндовс)
    ls -a Показывает все файлы и директори находящиеся в данном каталоге
    ls -l Расширенные (подробные) свойства каталогов (в виде списка)
    ls -a -l или ls -al Выдет полный подробный список (сочатает в себе 2 команды)
    ls Рабочий\ стол/folder/ Находясь в любой дериктории, данной командой можно получить сведенья о файлах по задаваемому адресу (Рабочий\ стол/folder/), в данной ситуации folder не входя в нее командой cd

    cd - Переход в последний каталог !!!в котором был до перемещения!!!
    cd Видео Переход в каталог видео

    hmod 777 text.txt
    chmod 777 *
    chmod -R 777 *     		Инструмент для повышения безопасности ОС, который позволяет назначить права доступа файлам или каталогам.
    777 это -rwxrwxrwx
    666 это -rw-rw-rw
    744 это -rwxr--r--
    700 это -rwx------
    644 это -rw-r--r--
    640 это -rw-r-----
    444 это -r--r--r--
    755 это -rwxr-xr-x
    505 это -r-x---r-x
    rwx -означает r - read чтение w - write запись x - executable исполняемый.

    Задает команду для всех файлов в каталоге.

    Рекурсивно (-R обходить рекурсивно, то есть не только файлы текущей директории но и все вложенные каталоги и файлы) задает права доступа всем файлам текущей директории и всем вложенным каталогам и файлам в нее.

    Создание директорий/файлов
    mkdir new Создает каталог с любым именем (в данном случае «new») в текущей директории
    mkdir -p new/new2 Создает древо директорий
    > file.txt Создает файл с названием file в текущей директории
    echo "!" > table.html Создает файл table.html с «!» внтури
    > new/file.txt Создает файл с названием file в заранее созданной папке new !!!!! не создает папку!!!!
    mkdir -p ./f1/f2/f3 && echo "sometext.." > ./f1/f2/f3/file.txt Создает древо дерикторий, в конечном каталоге которого создает файл file.txt с текстом внутри sometext..
    mkdir -p ./my_web/css_img/images && echo "!" > ./my_web/html.html && > ./my_web/css_img/css.css Создает древо дерикторий с файлами внутри (html.html, css.css), в файлe html создаем текст внутри (восклицательный знак !).
    Подробнее:
    mkdir -p ./my_web/css_img/images - создает древо директорий
    echo "!" - добавляет символ воскл. знак (возможно любой текст) текст в файл по пути > ./my_web/html.html
    и дальше добавляем еще один файл без текста в данную директорию && > ./my_web/css_img/css.css

    Поиск файлов, слов в файлах
    grep «текст в файле» text.txt Команда ищет заданную строку в заданном текстовом файле.
    grep текст text.txt Команда ищет заданное слово в заданном текстовом файле.
    grep «текст в файле» -rn Команда ищет заданную строку в заданной директории.
    -r — grep обойдёт каталог рекурсивно
    -n — grep выведет номер строки в результатах
    grep -rn какой /home/dolor/Рабочий\ стол/folder/ Команда рекурсивно ищет слово «какой» в заданной директории
    какой — указываем слово, которое ищем
    / home/... — указываем директорию.
    grep -rn какой /home/ Например /home/ и будет искать по абсолютно всем файлам вложенным в данный каталог
    find Рабочий\ стол/folder/*.txt

    echo Команда echo - это очень простая и в то же время часто используемая встроенная команда оболочки Bash. Она имеет только одно назначение - выводить строку текста в терминал, но применяется очень часто в различных скриптах, программах, и даже для редактирования конфигурационных файлов.
    echo Какой-то текст Выводит в терминале текстовую строку вписанную после команды, в данной ситуации "Какой-то текст".
    echo Какой-то текст > file.txt Запись текста «Какой-то текст» из консоли в текстовый файл
    ls -l >file2.txt Записывает выводимый командой ls -l текст в уже существовавший файл с названием file2.txt !!!!!!если в файле была какая то информация то ее сотрет и перезапишет этим текстом!!!!
    ls -l >>file2.txt Добавляте в файл свои данные не стирая уже ранее имевшиеся данные
    echo «lol» | tr «l» «w» Команда заменяет букву из слова «l» на нашу новую букву «w» и выводит текс с произведенной заменой !!! l и w можно прописывать без кавычек!!!
    echo "LOL" | tr [:upper:] [:lower:] Перегоняет верхний регистр в нижний или наоборот при изменении команды.
    sort text.txt Сортировка текста по алфавиту из файла, в терминале, но без изменений в файле, при желании можно его перезаписать но уже другой командой
